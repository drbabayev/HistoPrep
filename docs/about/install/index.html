Install

HistoPrep uses OpenSlide to read slide images, which unfortunately requires that the original C++ library is installed on your system. The installation instructions can be found here.

Then you can simply run.

pip install histoprep

There is an annoying bug, which sometimes pops up when installing OpenSlide. This happens if OpenSlide is shipped with an old version of pixman library. There are three ways of fixing the problem.
